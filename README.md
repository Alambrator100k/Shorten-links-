# 🔗 موقع اختصار الروابط والباركود - خدمات متعددة

أداة ويب متكاملة لاختصار الروابط باستخدام ثلاث خدمات مجانية مختلفة (TinyURL، Bitly، is.gd) مع إمكانية إنشاء باركود QR يعمل بشكل دائم.

✨ المميزات الرئيسية

🔗 ثلاث خدمات لاختصار الروابط

· TinyURL: خدمة مجانية وسريعة بدون تسجيل
· Bitly: أشهر خدمة اختصار روابط مع ميزات متقدمة
· is.gd: خدمة بسيطة وسريعة بدون إعلانات

📱 ميزات الباركود

· إنشاء باركود QR: توليد باركود فوري للرابط المختصر
· يعمل بشكل دائم: الباركود يعمل حتى بعد مغادرة الموقع
· تحميل الباركود: حفظ الباركود كصورة PNG
· جودة عالية: دقة عالية للباركود مع إمكانية المسح بسهولة

🛠️ أدوات إضافية

· نسخ الرابط: نسخ الرابط المختصر إلى الحافظة
· مشاركة الرابط: مشاركة الرابط عبر التطبيقات المدعومة
· إنشاء جديد: بدء عملية جديدة بسرعة

🎨 تصميم عصري

· واجهة جميلة: تصميم متدرج الألوان مع تأثيرات أنيقة
· متجاوب بالكامل: يعمل على جميع الأجهزة (جوال، جهاز لوحي، حاسوب)
· تفاعل سلس: تأثيرات حركية وتفاعلية

🚀 طريقة الاستخدام

1. اختيار الخدمة

· اختر إحدى الخدمات الثلاث المتاحة (TinyURL، Bitly، is.gd)
· ستظهر معلومات عن الخدمة المختارة

2. إدخال الرابط

· أدخل الرابط الطويل في حقل الإدخال
· تأكد من صحة الرابط (يبدأ بـ http:// أو https://)

3. إنشاء الرابط المختصر والباركود

· اضغط على زر "اختصِر الرابط وأنشئ الباركود"
· انتظر لحظات حتى تتم المعالجة

4. عرض النتائج

· سيظهر الرابط المختصر
· سيظهر باركود QR للرابط
· يمكنك استخدام الأزرار الإضافية

📊 مقارنة الخدمات

الخدمة المميزات العيوب
TinyURL سريع، مجاني، بدون تسجيل روابط طويلة نسبياً

🔧 التخصيص

تغيير حجم الباركود

```javascript
// في ملف JavaScript
new QRCode(document.getElementById('qrcode'), {
    width: 300,  // غيّر القيمة
    height: 300, // غيّر القيمة
    // ...
});
```

تغيير ألوان الباركود

```javascript
new QRCode(document.getElementById('qrcode'), {
    colorDark: "#000000",    // لون النقاط
    colorLight: "#ffffff",   // لون الخلفية
    // ...
});
```

إضافة خدمات إضافية

```javascript
// أضف خدمة جديدة في المصفوفة
const services = [
    { id: 'tinyurl', name: 'TinyURL', api: '...' },
    { id: 'bitly', name: 'Bitly', api: '...' },
    { id: 'isgd', name: 'is.gd', api: '...' },
    // أضف خدمتك هنا
];
```

⚠️ ملاحظات هامة

1. Bitly API: يستخدم مفتاح API عام للتجربة، قد يتوقف عن العمل في أي وقت
2. CORS: بعض الخدمات قد تواجه مشاكل CORS، نستخدم بروكسي لتجاوزها
3. مدة الباركود: الباركود صالح بشكل دائم طالما الرابط المختصر يعمل
4. خصوصية الروابط: الروابط المختصرة عامة ويمكن لأي شخص الوصول إليها

🐛 استكشاف الأخطاء

المشكلة: فشل في اختصار الرابط

· الحل: تحقق من صحة الرابط، جرب خدمة أخرى، تأكد من الاتصال بالإنترنت

المشكلة: الباركود لا يظهر

· الحل: تأكد من تمكين JavaScript، حاول تحديث الصفحة

المشكلة: مشكلة في النسخ

· الحل: استخدم Ctrl+C يدوياً، تأكد من منح الإذن للحافظة

📚 أمثلة للاستخدام

استخدام الباركود

1. اعرض الباركود على شاشة جهازك
2. استخدم تطبيق قارئ الباركود على هاتفك
3. امسح الباركود للوصول إلى الرابط المختصر

مشاركة الروابط

1. اختصر الرابط
2. استخدم زر المشاركة أو انسخ الرابط
3. أرسل الرابط عبر وسائل التواصل
